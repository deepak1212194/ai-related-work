Get started

Run any project in 3 steps

Each project is self-contained. The pattern is the same for all five.

1

Clone the repo

Pull the full portfolio in one shot.

2

Pick a project, install

Each folder has its own requirements.txt — no shared state.

3

Run the entry script

Each project README documents the exact command.

git clone https://github.com/deepak1212194/ai-related-work.git
cd ai-related-work/01-rag-faq-bot

python -m venv .venv
.venv\Scripts\activate          # Windows
source .venv/bin/activate       # macOS/Linux

pip install -r requirements.txt
python -m src.ingest
python -m src.pipeline -q "What is FAISS?"
